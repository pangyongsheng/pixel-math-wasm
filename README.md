# Pixel Math WASM

图像处理与数学算法实战平台，基于 Rust + WebAssembly 构建高性能像素级运算。

## 项目矩阵

30 个可独立运行的小项目，从单像素滤镜到 2D 粒子物理引擎，最终合成一个 **WASM 图像处理数学工具站**。

---

## 第一部分：单像素画笔

> **给单像素上色**——从灰度化到单色滤镜，最基础的图像处理工具集。


| # | 项目               | 简介                                       | 涉及技术                              |
| - | ------------------ | ------------------------------------------ | ------------------------------------- |
| 1 | 图片灰度化处理     | 实现加权灰度算法，彩色图转黑白灰度图       | RGBA 通道拆分、数值加权运算、像素遍历 |
| 2 | 图片反色/负片效果  | RGB 通道 255 反向，生成底片风格            | 单像素数值映射、基础循环遍历          |
| 3 | 亮度/对比度调节器  | 滑块控制，实时增减亮度、拉伸压缩对比度     | 浮点数值映射、区间归一化              |
| 4 | 二值化黑白阈值分割 | 设定阈值，图片只保留纯黑、纯白两色         | 条件判断、像素二分类                  |
| 5 | 基础单色滤镜       | 复古泛黄、冷色调、红通道增强、色相简单偏移 | RGB 通道独立调控、色彩线性偏移        |

## 技术栈

- **Rust** — 高性能计算核心
- **WebAssembly (WASM)** — 浏览器端高效运行
- **wasm-bindgen** — Rust 与 JavaScript 交互
- **Canvas API** — 图像渲染与可视化

## 项目结构

```
pixel-math-wasm/
├── src/              # Rust 源码
├── pkg/              # WASM 编译产物
└── README.md
```

## 开发说明

```bash
wasm-pack build --target web
```

编译后的 WASM 模块可在浏览器中直接调用，实现高性能像素处理与数学运算。
